## Tourism Recommendation System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/12856081959244b7b27b3f8d11e15575/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Forum
5. Attraction Information
6. Travel Routes
7. Announcement Information
8. Personal Center
9. Attraction Favorites
10. Route Favorites
11. Backend Management


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Basic Data Management (Attraction Type Management, Route Type Management, Announcement Type Management)
5. Forum Management
6. Attraction Information Management (Attraction Information Management, Attraction Information Comment Management, Attraction Information Favorite Management)
7. Travel Route Management (Travel Route Management, Travel Route Comment Management, Travel Route Favorite Management)
8. Announcement Information Management
9. User Management
10. Carousel Information
